Privacy Policy

Thank you for using pdf2png — a locally-run PDF to PNG conversion tool. We take your privacy and data security seriously.

1. Information We Do Not Collect

This software is a fully offline desktop application and does not collect any personal information or file content, including but not limited to:

  • Your name, contact information, or email address
  • Your IP address, location, or device information
  • The content of PDF files you open or the converted PNG files
  • Your usage records, operation behavior, or access paths

2. File Processing Method

All file operations are performed locally on your device. We do not upload, transfer, or store any of your file content.

  • The software only reads PDF files you actively select
  • The conversion process is entirely local and does not require an internet connection
  • The output PNG file is saved only to the local path you specify

3. Network Connection

This software runs offline and does not rely on any network connection. The software includes the following feature solely to enhance user experience:

  • Update Check: Connects to an official server to check for new versions (no data is uploaded)

The software does not communicate with any server remotely.

4. Third-Party Libraries

All libraries are used locally and do not involve data uploads or remote access. They will not affect your privacy.
